Master Cheat Sheet – Recommended Settings
XTTS v2 – Daily Driver (GPU or CPU)

Fast • Multilingual • Reliable • Built-in voice cloning

Parameter Recommended Notes
Temperature 0.65 – 0.75 >0.85 → slurring, <0.55 → robotic
Speed 0.90 – 1.15 Rubberband handles extremes
Repetition Penalty3.0 – 6.0 Too low → stuttering on long text
Top P / Top K Default Only change if gibberish appears
Whisper Tolerance 78 – 84% >90% too strict, <70% lets garbage through

Pros: Fast, CPU-friendly, any language
Cons: Voice cloning good but not perfect

Fish Speech 1.4 – Highest Quality Cloning (GPU only)

Best voice similarity • Very high quality • Slow inference

Parameter Recommended Notes
Temperature 0.60 – 0.70 >0.75 → unnatural accent drift
Top P 0.80 – 0.90 Avoid below 0.70
Reference audio 8–20 s cleanNo music or background noise
Whisper Tolerance 70 – 80% Can be stricter than XTTS

Pros: Near-perfect cloning
Cons: 3–8× real-time (4090), occasional crashes (use ##recover##)

Kokoro 82M – Fast CPU Inference

Extremely fast • Good quality • Skips unknown words

Parameter Recommended Notes
Speed 1.3 – 1.6 >1.7 → chipmunk
Pitch / Energy Adjust freelyBig impact on expression
Whisper Verify Off or ≥92%Lower values fail often

Pros: 30–40× real-time on CPU
Cons: Unknown names → silence

Stable Audio Open 1.0 – SFX & Short Music (≤47 s)

Best-in-class sound effects • Short musical clips

Parameter Recommended Notes
Length 5–12 s (SFX)

Golden Rules & Essential Tips
  • • Leave save path blank → preview only (no file saved)
  • • Enter folder name → auto-saves to projects_output/ThatFolder
  • • ##recover## → instantly resumes crashed job
  • • Whisper keeps failing? Edit job.json → set "verify_whisper": false
  • • Kill stuck job → close the app entirely
  • • Need true concurrency → run multiple instances on ports 5006–5009
  • • Post-processing (Rubberband, EQ, etc.) is required for professional results
